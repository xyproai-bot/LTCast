# LTCast press kit

Everything a journalist, blogger, podcast host, or reseller needs in one place. The goal is that a reviewer can write a full article without emailing us back.

## Contents

```
press-kit/
├── README.md                 You are here — start here
├── boilerplate.md            Short / long company and product descriptions
├── screenshots.md            Named list of screenshots to capture, with positions
├── logo-assets.md            Logo / icon export guide (sizes, formats)
├── press-contacts.md         Who to email, how to reach us, response SLA
├── attributions.md           Third-party assets, fonts, music, libraries
└── social-proof.md           Empty until launch — reviewer quotes go here
```

Binary assets (actual screenshot PNGs, logo SVGs, press-ready PDFs) are **not** committed to this repo. Keep them in a `press-kit/binary/` folder locally and upload to a public bucket for distribution (see `logo-assets.md` for the export pipeline).

Rule: if a file is >500KB or binary, don't commit it here. The text guide in this folder tells you how to produce it.
